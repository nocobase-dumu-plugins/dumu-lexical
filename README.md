# Nocobase wangeditor 富文本字段插件

### 注册安装

``` shell
cd packages/plugins
git clone git@github.com:nocobase-dumu-plugins/dumu-wangeditor-field.git
yarn
yarn pm add dumu-wangeditor-field
yarn pm enable dumu-wangeditor-field (可不执行，在插件管理中开启)
```
